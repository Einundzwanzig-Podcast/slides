# **Slide-Vorlage**

Präsentationsvorlage für [Einundzwanzig](https://einundzwanzig.space/) Vorträge, basierend auf [Marp](https://marp.app/).

---

# Aufbau der Slides

Einzelne Folien können mit `---` abgetrennt werden.

<!-- footer: Hier könnte dein Name stehen -->

```markdown
# Slide 1

Intro

---

# Slide 2

Outro

```
